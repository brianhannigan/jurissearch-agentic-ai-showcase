# Security Policy (Public Showcase)

## Scope

This repository is a **documentation and architecture showcase** for JurisSearch.  
It does **not** include production source code, production configurations, or live service endpoints.

## Reporting a vulnerability

If you believe you have found a security issue affecting the **production** JurisSearch system:

- **Do not** open a public GitHub issue.
- Report privately to: **[your security email here]**
- Include:
  - a clear description of the issue
  - steps to reproduce (if applicable)
  - impact assessment (what could an attacker do?)
  - any relevant logs/screenshots (redacted)

## What to expect

- Acknowledgement within a reasonable timeframe
- Triage and prioritization
- Coordinated disclosure if needed

## Safe Harbor

If you act in good faith and avoid privacy violations, data destruction, or service disruption, we will not pursue legal action.
